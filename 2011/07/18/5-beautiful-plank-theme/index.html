使用方法

要在Ubuntu中安装Plank界面，请运用以下的指示（将添加Plank/Docky测试PPA）：更新PPA版本库：更新列表ricotz/docky

sudo apt-get update
sudo apt-get install plank

安装plank

以上PPA可用于Ubuntu操作系统，给人梦幻般的感觉，同时又整洁、独特。

要选用其中一个主题，请将原Plank主题备份在~/.config/plank/theme/ 下，然后抽取已下载的主题到这个文件夹(~/.config/plank/theme/)，替换已有的文件。

注：~/.config/是根目录下的一个隐藏文件夹，按CTRL+H即可看到。
