最新20款Web设计和开发工具推荐

在这篇文章中，收集了最新20个Web设计和开发工具推荐给大家，这些工具都是免费的，不需要任何安装和更新的过程，感谢这些工具的作者为我们提供这么有用的工具。

1. X-Icon Editor

X-Icon Editor 是一款帮助你在线创建高分辨率图标的在线HTML5应用程序。

2. 520 Grid System

520 Grid System 是一个 HTML/CSS 框架，适合窄网站模板制作。

3. ClueApp

ClueApp 用一个有趣和简单的方法测试人们记住了你网站中什么内容。

4. ColorPix

ColorPix 是一款非常有用的取色工具。

5. ColorZilla

ColorZilla 是一款集合吸管、颜色选择、调色板等丰富功能的工具。

6. Font Comparer

Font Comparer 是一款字体比较工具，可以输入文字对比效果。

7. Generateprivacypolicy

generateprivacypolicy 这款工具可以帮助你创建专业的的用户隐私政策。

8. Gridulator

Gridulator 是一款易用的网格快速计算工具。

9. HTMLform

Htmlform 是一款很棒的表单制作工具，完全免费。

10. Launchlist

Launchlist 是一款站点发布清单工具，预设了28个需要检查的事项，也可以自定义。

11. Less Framework

Less Framework 是一个网格系统，用于设计自适应的网站。

12. PC Names

PCNames.com 用于及时检测.com, .net, .org, .info, .biz, .us, .mobi and .name等后缀的域名是否还可以注册。

13. Reset CSS

Reset CSS 可以重置任何内容的样式。

14. Rsizr

Rsizr 是一款在线图片缩放工具。

15. Spritebox

Spritebox 是一款所见即所得工具，帮助设计师从Sprite Image快速生成CSS代码。

16. ResizeMyBrowser

ResizeMyBrowser 是一款测试网站在各种尺寸下的表现情况的工具。

17. The Photoshop Etiquette Manifesto

Photoshop Etiquette Manifesto 帮助设计师更合理的组织 Photoshop 文件。

18. Tiny Letter

TinyLetter 让你以最简单的方式开始编写自己的邮件通讯。

19. Touchqode

Touchqode 是一款运行于智能手机上的代码编辑工具。

20. AngularJS

Angular 是一个可扩展的客户端HTML编译器和JavaScript框架，用于构建交互的WEB应用程序。
