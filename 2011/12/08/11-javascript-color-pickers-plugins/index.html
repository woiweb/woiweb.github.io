11个javascript拾色器插件

2011-12-08

几年前很难找到一款很好的拾色器，最近我已经看到很多发布的Javascript的拾色器插件，所以我决定把我找到的编辑起来，在本文中，我找到了11款Javascript的拾色器插件。它们中的一些相当复杂而其他的却相当简单，所以我确信总有一款是你所需要的。

ExColor

ExColor是一个jQuery插件，使用户可以从像Photoshop样的拾色器中选择颜色，它依附于input框并且可以通过简单的一单行函数进行调用。

JScolor

JSColor是一个非常容易实现的Javascript库，可以将任意指定的input转换成拾色器。这个拾色器是一个隐藏层，无需使用弹出窗口即可定位在input域的边缘上。

Advanced Javascript ColorPicker

ColorPicker是一个高级的可高度定制的JavaScript拾色器，而且不依赖于任何Js框架。它可以以6种不同颜色模式显示全部的颜色调色板。

Farbtastic

Farbtastic是一个jQuery拾色器插件，可以轻松的在任何网页中嵌入一个拾色器。这个拾色器依赖于一个存在的元素（例如，一个文本框）并且当选择一个颜色后会更新这个元素的值。

Javascript Color Picker

这个JavaScript拾色器拥有HSB和RGB选项，使用原型并且兼容所有主流浏览器。

mooRainbox

mooRainbow是一个极易整合的用mootools构建的JavaScript拾色器。这个拾色器可附属于任何一个在DOM中的元素，一个点击事件将会自动绑定那个元素，并且简单的点击它你将可以显示或隐藏mooRainbow。

ColorPicker

一个简单的组件可以让你用在Adobe Photoshop中同样的方式选择颜色。

jPicker

jPicker是一个快速的、轻量级的jQuery插件用于在你的web项目中引入一个高级的拾色器。

SimpleColor

jQuery Simple Color是一个非常简单的拾色器插件，它排列了一系列小方块用于选色。

Really Simple Color Picker

一个可预定义的拾色器，简单直接。

Flexi

Flexi是一个轻量级(4.3kb)的拾色器，它不使用任何图片（感谢SVN/VML），CSS而仅仅是Javascript。它是一个独立的脚本不需要任何Javascript框架，并且能将选择的颜色转换成16进制，HSV和RGB模式。
