开发与设计网站的优秀开源软件

2010-12-12

写篇文章列出开发与设计网站的优秀开源软件真是不容易但却相当重要。这不是因为没多少软件可以选择，事实上服务器端软件、内容管理平台、向量绘图工具等应用程序都很多。提供免费开源的产品是很令人感动的事，这篇文章会帮你找到真正对你每天都有帮助的一些软件。毫无疑问，喜欢如下软件的读者会和不喜欢的读者一样多，他们会有自己所喜欢的一些软件。欢迎读者提出自己的建议和看法，相关并且合适的建议会被加入到这个列表中。

Joomla

Joomla
Joomla是最受欢迎的开源CMS系统之一并且每天有上百万的网站在使用它。使Joomla脱颖而出的是它庞大的社区和无穷多的免费主题和组件。使用Joomla的方式可以有无数种，加上Virtuemart的Joomla就会变成一个功能齐全的商店。

Drupal

Drupal 是个非常流行的CMS平台，类似于WordPress和Joomla。它可能没有另外两个CMS那么多高质量的主题可用，但是Drupal在良好的技术设计和专注于可维护性方面是独一无二的，这是它被许多人偏爱的原因。Drupal对配置设置做得相当深入细致而且允许以独特的方式参与社区。

WordPress


WordPress 是世界上最流行的博客平台，使用它的网站惊人地多达2亿2百万个。你可以随心所欲地使它变得简单或者复杂，WordPress支持非常多的插件，可以把一个标准的博客变成任何你想要的样子。除了博客，WordPress能够直接用作内容管理系统以驱动从电子商务站点到社会化网络的任何网站。

MySQL


MySQL 可能是最流行的数据库了，并且它还开源甚至完全免费。MySQL企业版本不免费但比起其它的商用企业级数据库来说这个价格只是象征性的，并且技术支持仍然是世界级的。免费的MySQL版本无处不在而且充当着上述流行的CMS系统的基础。大多数MySQL用户知道可以通过著名的phpMyAdmin来完全控制数据库。去年的早些时候，MySQL的用户群震惊于Oracle对MySQL的收购（做为接管SUN公司的一部分），并且担心MySQL不明朗的前景。 Oracle感到了压力并在2009年12月的新闻稿中公开回应了相关的问题。

Apache webserver

Apache httpd自从1996年4月就已经是Internet上最流行的服务端软件了。Apache免费、稳定、运行良好的同时还极其灵活并且功能强大。所有使用WordPress和Joomla等平台的著名且受欢迎的博客和站点都运行在Apache上，这类网站数以亿计。Apache在企业基础设施中也很受欢迎，在端口和B2B网关等前面用作软件反向代理组件。Apache的强项来自于它的组件系统和像mod_rewrite这样强大的处理URL的功能组件。

Apache Tomcat


Apache Tomcat是一个由Java Servlet和JavaServer Pages技术实现的开源软件。Apache Tomcat驱动了许多不同行业和组织的大规模的网站应用程序。Tomcat与Apache web server的不和谐之处在于Apache是一个由C实现的HTTP web server。Tomcat是一个免费的轻量级容器，可以服务于Java网络项目。

Magento


Magento是世界上最快的电子商务平台，被3万多个商家使用，包括三星、奈斯派索
(Nespresso)和乐斯菲斯(The North Face)。Magento共享版在开源软件协议下可以免费使用。企业版则必须付费，提供了诸如多个商店、商店信用和礼品券等功能。

Aptana

Aptana 是在Eclipse上开发的，已经是个很成熟的开源项目了。Aptana很受欢迎并且下载量已经超500万次。它支持html、css和 javascript，编辑方便自然，是个真正有用的站点管理器。它有很多用于PHP、Ruby、JavaScript和Adobe Air的插件。它甚至有iPhone专用的开发环境。Aptana紧密地整合进了他们的云主机服务中。

Netbeans

Netbeans 是个免费开源的集成开发环境。它开始时是个纯JAVA的IDE，但现在它支持广泛并功能丰富。可以用它提供的工具开发专业的桌面级、企业级、网站级和移动电话上的应用程序，既可以使用Java也可以使用C/C++、PHP、JavaScript、Groovy和Ruby等语言。Netbeans以启动速度快和强大的调试能力著称。

Notepad++

可靠性强、可定制、体验良好并且免费，Notepad++是最流行的Windows下的优秀文本编辑软件。它功能丰富，包括：语法高亮；对CSS, C, C++, C#, Java, JavaScript, SQL, HTML, XML和PHP等语言可折叠显示和自动完成代码；多文件视图；标签编辑；缩放显示；书签功能等。

Inkscape

Inkscape 是个跨平台的向量图形应用程序，可能是类似可用的开源软件中的最佳选择。它可以用来创建专业的图形、Logo、图表、图标、地图等很多东西。它提供了同类商业软件中的所有功能，并更加注重用户友好度，同时仍支持高级SVG (Scalable Vector Graphics )功能(markers, clones, alpha blending, etc.)。这个工具也允许用户导入很多标准格式的文件，如JPEG, BMP, PNG, EPS and TIFF，并输出PNG格式文件。

Mozilla Firefox


Firefox目前的浏览器市场占有率是24.43%，但这个数字还在增长。Firefox的用户中专业人士比较多，它的广泛使用归功于它有着大量的第三方附加组件，这些组件让用户能够修改他们的浏览器的任何一个微小的细节。

FileZilla


FileZilla是个很成功的跨平台FTP客户端。它也可以做为服务器端软件，但只能用于Window环境。作为一类由Tim Klosse在2001年1月开发的项目，FileZilla已经在SourceForge.net网站上总下载量排名第五了。

XAMPP

很多人都经历过安装Apache的繁琐经历，如果加上MySQL, PHP 和 Perl那就更困难了。XAMPP改变了这一切，它能够简单地搭建出包含MySQL, PHP 和 Perl的Apache服务端。XAMPP对快速建立一个本地的开发环境十分必要。XAMPP支持多个系统：Windows, Linux, OS/X, Solaris等。

BIND


BIND显然是Internet上应用最广泛的DNS软件。它提供了一个健壮稳定的平台，在这个平台上可以建立完全遵从DNS标准的分布式计算系统。它是域名解析系统协议的一个参考实现，但它也是一个产品级的软件，适合在高容量和高可靠性应用中使用。
