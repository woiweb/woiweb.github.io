微软Windows Phone7智能手机系统七大功能亮点

2010-10-12

2010年10月11日，微软公司正式发布了最新智能手机操作系统Windows Phone 7，并展示了十款采用Windows Phone 7的智能手机，首批Windows Phone 7手机分别来自HTC、LG、三星和Dell等品牌，共有10款不同型号的产品，其中九款最早将于10月21日上市。

Windows Phone 7包括人际、图片、游戏、音乐与视频、办公和商店内容，同时还整合了许多受欢迎的微软客户服务，如Xbox LIVE、Windows Live、Bing和Zune。所有 Windows Phone 7手机都将采用高通 (Qualcomm) 的高性能Snapdragon处理器。

微软认为，Windows Phone7的目标非常明确，就是将人们真正需要的东西确实整合在一起，放在用户面前，让他们能迅速完成操作或是沉浸在他们一直在寻找的应用体验中。

Windows Phone7手机的界面可以说是很多功能的交汇中心，例如人际交流、音乐和视频、照片、游戏和办公。无论用户在手机中深入多少界面，这些交汇中心都不会再远离你。例如，用户中心里面，不仅提供了更明确的联系信息、电话号码，也会显示朋友们刚刚在Facebook上更新的状态。用户可以呼应朋友们在SNS网站上的信息或是直接在用户中心里给他发一个短信，不用像以往那样还要开启一个特定的应用。这个中心还会及时更新，带来照片和信息，因此，大部分情况下简单的一瞥或是几下点击就可以用户了解电话、邮件信息，以及朋友和同事们正在做什么。

下面就是微软总结出的Windows Phone7七大功能亮点：

启动屏幕

Windows Phone7的启动屏幕把对手机用户最重要的应用和功能放在前台，让应用明确而容易操作。

用户中心：朋友和联络信息，在移动中连接

用户中心针对人们当今交流的方式而生，带来了朋友们的Facebook状态更新、各种联系信息和电话号码。在这里，用户可以回应朋友们的SNS信息或是直接发送短信。

游戏中心：集成Xbox LIVE

游戏中心不仅仅是一个游戏的文件夹，这是一个能访问游戏、寻找新游戏和连接朋友们的地方。它延展了Xbox LIVE社交游戏体验，并提供了只有Windows Phone7才有的移动游戏感受。

移动中的生产力：办公中心

微软办公中心提供了大家熟悉的微软Office应用的移动版本，包括Microsoft Word Mobile、Excel Mobile和PowerPoint Mobile，可以浏览和编辑文档。用户还可以通过Microsoft SharePoint Workspace Mobile 2010访问各种信息，实现协同办公。

照片中心：拍摄并分享照片

Windows Phone 7包含一个独立的相机按钮，即使手机处于锁定状态也可以使用，帮助用户抓住关键瞬间。Windows Phone还让分享照片变得更简单，通过照片中心可以分享和评论自己与别人的照片。

音视频中心：

音视频中心把所有的媒体放在一个地方，包括歌曲、视频和广播。用户可以听调频广播或是从互联网上下载、聆听音乐。Zune市场也是这个体验的一部分，让用户在移动时也可以了解最新的歌曲、电影，并和朋友们分享。

本地搜索：集成必应搜索

Windows Phone 7带来了整合的必应搜索体验，内置GPS和必应地图。本地搜索可以迅速提供相关场所的电脑号码、价格、评价和位置。用户可以选择驾车或是徒步路线，观看街景图像和实时交通状况。
