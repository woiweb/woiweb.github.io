15款WordPress用户的实用工具

2010-08-07

如果你使用开源博客平台WordPress，你会发现网上有非常多的工具可以助你提高工作效率，确保你的WordPress站点顺利运行。这篇文章将聚集这些非常方便的应用程序、插件以及浏览器扩展来帮助你完成各种博客任务，涉及的范围涵盖简单的抓屏到WordPress站点的安全。

1. ScribeFire**将Firefox浏览器变为一个功能齐备的博客编辑器**

Firefox的插件ScribeFire为你写博客文章提供了一组非常实用的功能和选项。你可以拖放多种来源的内容(WordPress可视化编辑器目前并不支持此功能)、管理博客文章并给文章做注释、通过内置的FTP上传文件、管理上传的图片。ScribeFire是对WordPress内置编辑器有更高要求用户的理想选择。

2. WordPress Helper**给你提供了实用的提示帮助和功能**

不管你是WordPress新手还是经验非常老道的博主，你将会觉得WordPress Helper 是一个备受欢迎的助手。WordPress Helper 将会通知你新的WordPress 版本，让你可以自定义键盘快捷键，并且还提供了帮助文档，只要用鼠标轻轻一点该页面就可获得帮助。

3. Screen grab to WordPress!**轻松截屏**

如果你的文章中包含很多截图，你应该试试免费的Firefox插件，Screen grab to WordPress!。你可以将一个页面保存为图片，然后使用该工具点击一个按钮就可以自动将图片上传到你的WordPress站点。

4. WordPress Scanner**让你的博客安全无忧**

有了WordPress Scanner这款工具，你就可以高枕无忧了，因为你无需担心博客会遭受任何潜在安全漏洞的攻击。WordPress Scanner 有基于web的应用版本和WordPress插件.版。

5. Broken Link Checker**确保文章没有死链接**

WordPress插件，Broken Link Checker 会负责检查你的链接看是否是有效链接。它通过扫描你的博客文章和页面，如果发现了任何死链接就会通过WordPress管理后台通知你。并且你还可以设定扫描的时间，建议你挑选一个访问量比较低的时间段扫描。

6. Google Syntax Highlighter**让你轻松地在文章中插入代码**

如果你曾经尝试过通过WordPress可视化编辑器在文章中插入代码，结果却发现代码被整的不成样子，你就会爱上Google Syntax Highlighter插件。它可以帮助你在WordPress文章和页面格式化代码。有了Google Syntax Highlighter ，你想在文章中显示代码变得更加容易，并且利用颜色来突出句法，读者阅读起来也更加轻松。如果你觉得不太喜欢Google Syntax Highlighter，你还可以使用类似的几款插件，WP-Syntax, WP-Chili,或者 SyntaxHighlighter Plus。

7. Insights**改善信息搜索**

WordPress插件Insights ，协助你寻找和收集信息和媒体资料，为你写文章提供有用参考。你可以通过Insights搜索针对特定话题的所有有关博客文章，它为寻找外部站点和服务，诸如Flickr, Youtube, Wikipedia,和 Google Blog search 提供了非常直观的界面。

8. Windows Live Writer**，让你可在脱机状态下写博客**

Windows Live Writer是微软公司的其中一个非常实用的应用。Windows Live Writer 是一款免费的桌面应用，兼容WordPress (以及其他流行的博客平台，如Blogger和 Typepad)。 有了这个应用，你可以在脱机状态下写博客文章，并且随时可发布。
如果你是Mac用户，请使用 ecto 或者 Thingamablog (跨平台的桌面应用，适用于Linux, Mac和Windows用户)。

9. Revision Control**非常适合多作者博客**

如果你的博客有多名作者和编辑，或者你经常写篇幅很长的文章，这篇文章需要花上几个小时（甚至几天）来完成，你将会发现 WordPress插件Revision Control 非常实用。通过Revision Control，你可以轻松地管理一篇文章或页面的不同版本，这样你可以对不同版本进行备份还可在不同版本间还原。

10. Organize Series**管理不同系列的博客文章**

有时候，特别是一些比较大的话题，压缩到一篇文章显得不太合适时，你会需要在不同日期将文章分部分发布，WordPress插件Organize Series 是一款非常实用的插件， 它为你的“系列文章”提供了非常多的选择，诸如，能够列出此系列下的其他文章，并且给特定系列添加了指定图标以方便识别。

11. WordPress Comment Moderation Notifier**加快评论审阅**

写博客一个非常重要的组成部分就是管理读者的评论并及时对评论做出回复。WordPress Comment Moderation Notifier 栖身于你的系统托盘中，当有新的评论出现时，它就会通知你。根据你所做的设置，(它在设定的时间间隔内检查是否有新的评论),这样你就可以更加及时地管理和回复评论，这将会让你更受读者爱戴。

12. WordPress Comment Ninja**扩展评论审阅的选项**

WordPress团队确实在评论的审阅和管理上有了改进。 如果你还是不满意，试试WordPress Comment Ninja,，是一个免费的 Greasemonkey 脚本 ，它扩展了评论审阅和回复的选择。例如，你可以通过电子邮件回复评论或者将评论逐字转发给某人。使用WordPress Comment Ninja，你需要使用Firefox浏览器和Greasemonkey插件。

13. AdServe**管理WordPress中的广告**

如果为了赚钱或者是为了分担服务器的费用，你在博客中投放了赞助商的广告，AdServe插件将会帮助你直接在WordPress中管理这些广告而无需借助第三方脚本或网络服务。AddServe极高的可配置性将会帮助非常有效地管理广告。它还会跟踪并显示广告相关的统计数据，包括页面展示次数和点击率。

14. Simple Tags**使标签更加完美**

如果你经常给文章加标签，可能需要使用WordPress插件 Simple Tags，它拓展了WordPress中的标签功能。帮助你管理关键词，根据你的文章内容给出相关关键字的提示 (使用免费的网络服务Yahoo Term Extractions API)，并且允许你批量地编辑和替换现有的标签。

15. Twitme**自动对新发布的文章在Twitter上发布**tweets

Twitter已经迅速成为了非常有影响力的网络应用。如果发布新文章时，你想自动通知Twitter上的followers，请使用WordPress插件Twitme，通过它，每当你发布新的博客文章时，就会立刻发布一个 tweet在 Twitter上。或者你还可以试试其他插件如Twitter for Wordpress, Twitter Tools, Twitpress, 和SimpleTwitter，看看哪一个最适合你。
