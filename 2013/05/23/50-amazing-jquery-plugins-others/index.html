50个令人惊奇的jQuery插件（杂货篇）

本篇所述的插件和库都不属于其他栏目，不过确是很值得一看，内容倒计时，支持增、删、改、查的列表，下拉刷新，图表，地图等。

42. jQuery Countdown

jQuery Countdown (demo) 是一个对于一件事显示剩余天数，时，分和秒的插件，并且通过每秒会用动画更新剩余时间。它简单易用，引入插件的JS和CSS文件，并在document完成时调用它：

$(‘#countdown’).countdown({
    timestamp   : new Date(2015, 0, 3) // January 3rd, 2015
}

倒计时将放置于#countdown的div中。

jQuery Countdown

43. List.js

List.js (github) 是一个增强纯HTML列表属性的插件，它可搜索，排序和过滤列表里面的条目。你能动态地增，删，改这些条目，List.js是独立的不依赖jQuery或其他库。

List.js

44. jQuery PointPoint

jQuery PointPoint (demo) 是一个帮助你吸引用户注意力到页面特定部分的插件，它在鼠标光标旁放置一个小箭头，这对露掉的表单，需要点击的按钮或需要滚动到可视范围的验证错误非常有用。

jQuery PointPoint

45. Social Feed

Social Feed (github) 是一个在你社交帐号下展示最新的公共更新信息的jQuery 插件，它拥有简洁的设计我们可通过修改模板实现自定义。

Social Feed

46. Hook.js

Hook.js (github) 是一个有趣的实验，它给web带来你常在移动应用上看到的“下拉刷新”的特性。你可使用这个来用新内容更新你的移动站点 ，获取新文章或更多。

Hook.js

47. jQuery PercentageLoader

PercentageLoader (bitbucket) 是一个进度条插件，它使用HTML5 canvas实现丰富的图表外观，js文件仅10kb（压缩后）。

Loader.js

48. Chart.js

Chart.js (github) 是一个新的图表库，它使用HTML5 canvas实现并在IE7/8提供回调函数，该库支持6种动画的可定制图表类型 。 Chart.js 没有任何依赖且大小仅5。跟Flash说白白吧！

Chart.js

49. Tooltipster

Tooltipster (github) 是一个强大的显示提示条的插件，它可工作于桌面和手机浏览器，给你对内容完全控制且支持很多回调函数，所以你可在你的应用里调用。如果你需要一些更轻量级的并且根本不使用JS，你可以尝试 hint.css.

Tooltipster

50. Toolbar.js

Toolbar.js (github) 是一个显示整齐弹出菜单的插件，你可监听事件的行为和选择菜单中的图标，这对移动web应用界面很有帮助。

Toolbar.js

51. gmaps.js

gmaps.js (github) 是如此有用的类库，我决定把它加到以上的50个插件中，Gmaps是一个使使用Google地图变得更加简单的强大的类库，你可以放置标记，监听事件，设置座标，缩放级别有更多，所有用似jQuery一样的流式语法，这对比正规的地图JS API来说像呼吸新鲜空气一样，我极力推荐。

Gmaps.js
