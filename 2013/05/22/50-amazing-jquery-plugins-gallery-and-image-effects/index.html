50个令人惊奇的jQuery插件（画廊及图片特效篇）

2013-05-22

本章节中的插件目的是为提高访问者浏览你网站上的图片的方式，并且让你使用有趣的效果而使你的画廊引人注目。

33. jQuery TouchTouch

jQuery TouchTouch (demo) 是一款我们去年发布的一款插件，其目的是简化使用并且可用于移动设备，它使用CSS变换来使动画更平滑 ，且可动态地预加载图片。同时，它完全能通过修改一个独立的CSS文件来重新定制样式。

jQuery TouchTouch

34. iPicture

iPicture 是一个能创建交互的图片画廊的插件。它遮盖在图片上方便签显示在图片上侧，并可展示富文本内容像HTML，图片和视频等。它很容易集成到你的网站并且可通过扩展CSS来实现自定义。

iPicture

35. Adipoli

Adipoli (demo) 是一个用来创建流行图片悬浮效果的插件，有了它，你可以将图片变成黑白色，应用弹出效果，切片效果和盒子变换等，要使用这个插件，你只需引入必需文件并且定义开始和悬浮效果：

$(‘#image1’).adipoli({
    ‘startEffect’ : ‘normal’,
    ‘hoverEffect’ : ‘popout’
});

这将完美添加到你的网页中。

Adipoli jQuery Image Hover Plugin

36. Swipebox

Swipebox (github)是一 lightbox 插件，它支持桌面，移动设备和平板浏览器，它已经领会swip手势和键盘导航且视网膜， 使用方法：引入JS/CSS文件，并且添加swipebox样式到你想要在lightbox中显示的图片上。

<a href=”big/image.jpg” title=”My Caption”>
    <img src=”small/image.jpg” alt=”image”>
</a>

像本集合中剩余的插件一样，它能通过CSS完全自定义。

Swipebox

37. TiltShift.js

TiltShift.js (github) 一个使用CSS3的图像过滤器复制移轴效果的jQuery插件，这是一个概念的证明，目前只能在Chrome和Safari6。

TiltShift.js

38. Threesixty.js

Threesixty.js (github) 是一款创建可拖拽的360度视角的jQuery插件，你必须提供你的产品的套图（不同角度的拍摄）路径并且它将把这些图片整合到一个可交互的视图中，你可拖动或使用方向键来从不同角度浏览物体。这个和我之前写的那个全景展示插件jQuery3D全景效果展示插件 差不多，起码原理一样，呵呵。

Threesixty.js

39. Swipe.js

Swipe.js (github)是另一款响应式的幻灯片插件， 使得它脱颖而出的是，它是专门针对触摸设备的。这使得它不仅支持手势，而且也同样适应于本地应用程序。

Swipe.js

40. CamanJS

CamanJS (github) 是一个强大的图片操作库，建立在顶部的canvas元素上，有了它，你可以像用Photosho操作一张图片一样。我们几个月前用这个库构建了一个 web app with filters similar to Instagram。CamanJS用新的过滤器和插件是很容易扩展的，它配备了各种各样的图像编辑功能，且这将继续增长。它是完全独立的库并且基于NodeJS和浏览器工作。

CamanJS

41. SpectragramJS

Spectragram (github) 是一个用来让使用Instagram API 更容易的插件，它能抓取用户的feed，流行的图片或更多。

SpectragramJS
